PROJECTS

Wikipedia Viewer

This project is an app to view the first 10 related wikipedia articles to a search term. Each of the results will link to the Wikipedia page. There is also a button to use to go to a random Wikipedia article.

Random Quote Machine

The is a web app that will make a request to a quote API to give a random quote each time the random quote button is pressed. If there is a quote the user like they can use the tweet button to tweet that quote.

Local Weather

This application will get the user's location based on their IP address. It will then make a call to the Open Weather API to get the current weather conditions in their city. It will give a breif weather description, an icon image of this description, and the current temperature. The temperature can be changed between Fahrenheit of Celcius.
